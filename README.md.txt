# Compatibility Scorer

## Description
A rule-based Python script to score how compatible two roommates are based on:
- Sleep Time
- Cleanliness
- Work Schedule
- Food Habits

## Features
- Score between 0 and 100
- Loads mock profiles from CSV
- Ranks top roommate matches
